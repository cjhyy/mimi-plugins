# 动态视频提示词参考示例

> 以下示例均为本项目自有作品的原创范例，用于示范"导演讲戏 → 可执行视频提示词"的写法。
> 不依赖任何第三方模板库。每条都遵循本流水线的核心原则：叙事描述式、参考图已见、
> @引用明确用途、节拍密度匹配时长、头尾安全区。
>
> 注意：这些是**写法示范**，不是固定话术。实际提示词必须从导演讲戏本转化而来，
> 角色名以 production-bible 花名册为准，画幅以渲染规格为准。

---

## 示例1｜首帧图生视频 · 单镜头 · 8秒（古风玄幻）

**生成方式**：以一张分镜首帧图驱动，单镜头连续动作。

```text
以 @图片1 中的少年作为主角，场景参考 @图片5 的练功场清晨。
中景，少年独自立于空旷的练功场中央，灰蓝色的清晨微光从画面左侧斜射入场。
他缓缓抬起右手，黑色古朴戒指在晨光下泛出一点冷硬的暗光，
指尖微微一颤，一丝几不可见的暗红气息自戒面溢出，旋即收敛。
镜头从他的侧脸极缓推近至戒指特写，整体氛围沉静中暗藏张力。
音效：清晨微风、远处鸟鸣、一声极轻的气流嗡鸣与画面气息同步。电影感，冷寂内敛。
```

## 示例2｜首帧图生视频 · 分时段 · 15秒（古风玄幻，多镜头）

**生成方式**：单次生成 15 秒，按时间段精确控制；适合一段完整叙事弧线。

```text
以 @图片1 中的少年为主角，场景参考 @图片6 的觉醒仪式大殿。
0-3秒：远景建立，高大幽暗的石殿，少年立于殿心一束顶光之下，身影渺小，殿内静默。
4-8秒：中景缓推，少年闭眼凝神，眉头微蹙，掌心向上，一缕微光在掌中聚拢又黯淡，
        表情从期待转为一丝不甘。
9-12秒：特写他的侧脸，睁眼，瞳中映出殿顶幽光，低声自语："不甘心。"
13-15秒：镜头缓慢拉远，少年身影没入殿堂阴影，仅余顶光一束，画面定格。
音效：低沉殿堂回声、衣料摩擦、末段一记空灵钟鸣收束。氛围庄严压抑，仙侠高燃。
```

## 示例3｜续接上一段视频 · 向后延长 · 5秒（体育喜剧）

**生成方式**：承接上一段已生成的视频，向后延长；生成时长选"新增部分"的时长。

```text
将 @视频1 向后延长 5 秒，承接上一段大巴车驶入古城门的画面。
大巴车稳稳停下，车门"哐"地一声拉开，中年大叔陈梦第一个跳下车，
脚一沾地就叉腰抬头望向高耸的古城门飞檐，咧嘴一笑，露出一口白牙。
镜头从车门跟拍至陈梦半身，背后乡亲们扛着行李陆续下车，人声鼎沸。
全程跟拍流畅，画面阳光明媚、饱和度高。
音效：车门金属声、行李碰撞、乡亲们的方言喧闹、一段轻快诙谐的喜剧配乐起。轻喜剧燃爽。
```

## 示例4｜参考运镜 + 首帧 · 10秒（体育喜剧，宏大场面）

**生成方式**：以一张首帧图定画面，参考一段视频的运镜效果。

```text
以 @图片2 作为首帧（夜晚世界杯决赛球场全景），参考 @视频1 的航拍俯冲运镜。
0-3秒：高空航拍俯瞰，球场如一只发光巨碗嵌入夜色，顶部 LED 光环刺眼，绿茵场亮得泛白。
4-7秒：镜头俯冲下压，掠过红蓝两片球迷看台的色块海洋，人浪一圈圈翻涌，手机亮光如星点。
8-10秒：俯冲至球场边线上方掠过，草皮的翠绿与灯光的白在画面中飞速划过，定格于中圈。
音效：十万人声浪如低沉雷鸣层层叠起、解说员激昂前奏、末段鼓点重击。宏大、震撼、沸腾。
```

## 示例5｜首尾帧 · 8秒（古风玄幻，状态转变）

**生成方式**：@图片1 作首帧、@图片2 作尾帧，模型补全中间过渡。

```text
@图片1 作为首帧（少年立于崖边，神情落寞），@图片2 作为尾帧（少年眼神坚定，气息内蕴）。
中景固定偏微动，画面从首帧的萧索缓缓过渡：山风拂动少年额前碎发与衣摆，
他低垂的目光逐渐抬起，背脊一点点挺直，落寞神情收敛为沉静的坚定，最终定格于尾帧姿态。
光影由首帧的灰暗暖侧光，过渡到尾帧时一缕金光自远山斜照其面。
音效：山风渐起、衣料猎猎、末段一声悠远箫音。情绪由低沉转向蓄势待发。
```

## 写法要点回顾（从以上示例提炼）

- **开头先声明 @引用及其用途**：是首帧、尾帧、人物形象、场景、还是参考运镜，必须写明。
- **只写"变化"，不复述参考图静态内容**：人物长相、场景布局参考图里已有，重点写动作、运镜、情绪转变。
- **时长 ≥ 10 秒用分时段**：每段控制动作密度，段间衔接自然，遵守头尾安全区。
- **音频与画面同写**：背景音乐风格、环境音效、动作音效、台词（引号标注 + 语气）。
- **角色名 / 画幅 / 风格**：分别以 production-bible 的花名册、渲染规格为准，不在示例里写死。
